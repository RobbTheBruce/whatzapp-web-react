Este projeto é uma copia do Whatzapp Web usando React e NextJs.

## Instalação

Executar o comando do npm na pasta do projeto:

```bash
npm install
```

Execute o comando abaixo na pasta do index para iniciar o servidor:
```bash
npm run dev
```
